# Muru3d

3D modeling

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'muru3d'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install muru3d

## Usage

    $ bundle exec muru3d
